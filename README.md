# ATF Wines (Vite + Supabase)

Sitio estatico multipagina con Vite (`index.html` y `tactiq.html`) y formulario conectado a Supabase.

## Requisitos

- Node.js 18+

## Instalacion

```bash
npm install
```

## Variables de entorno

1. Copia el archivo de ejemplo:

```bash
cp .env.example .env
```

2. Completa:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

## Desarrollo

```bash
npm run dev
```

## Build de produccion

```bash
npm run build
npm run preview
```

## Tabla sugerida en Supabase

```sql
create table if not exists public.demo_requests (
  id bigint generated by default as identity primary key,
  name text not null,
  surname text not null,
  email text not null,
  message text,
  source_page text,
  created_at timestamptz not null default now()
);
```

Luego ajusta RLS/politicas para permitir `insert` desde el cliente anonimo (segun tu estrategia de seguridad).
